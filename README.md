# Personal Portfolio Page

## Overview

This project is a personal portfolio webpage for Maalik Watson. The page includes an About section, skills list, featured content, dynamically rendered project cards, a contact form, a welcome modal, and a dark mode toggle.

The purpose of this assignment is to practice using JavaScript objects, arrays, JSON, session storage, and DOM manipulation. The project information is stored as JavaScript objects, converted into a JSON string, saved in session storage, retrieved with JSON.parse(), and displayed dynamically on the page.

## Languages Used

- HTML
- CSS
- JavaScript

## Libraries Used

No external libraries are required for this project.

## Features

- Personal portfolio layout
- Skills list generated with JavaScript
- Three custom project objects
- Project array stored in session storage
- JSON.stringify() used to save project data
- JSON.parse() used to retrieve project data
- Dynamic project rendering with DOM methods
- Dark mode toggle using local storage
- Welcome modal
- Contact form alert message

## How to Run the Project

1. Open the project folder in Visual Studio Code.
2. Make sure `index.html`, `script.js`, and `README.md` are in the same folder.
3. Make sure the image files are inside an `images` folder.
4. Open `index.html` in a web browser.
5. Open Developer Tools.
6. Check session storage to confirm that `portfolioProjects` was saved.
